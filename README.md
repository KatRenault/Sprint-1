# Puppy Training Assistant

## Project Overview
The Puppy Training Assistant is an innovative application designed to help dog owners train their puppies effectively. It provides step-by-step training guides, tips, and tracking functionalities to ensure a comprehensive training experience.

## Features
- **Training Guides**: Detailed instructions for various training techniques.
- **Progress Tracking**: Users can track their puppy's training progress over time.
- **Tips & Tricks**: Helpful advice to deal with common puppy behaviors.

## Installation
To install the Puppy Training Assistant, clone the repository and install the required dependencies:
```bash
git clone https://github.com/KatRenault/Sprint-1.git
cd Sprint-1
npm install
```

## Usage
Run the application with the following command:
```bash
npm start
```
